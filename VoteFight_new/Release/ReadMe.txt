이동 : WASD
마우스 왼쪽 클릭: 공격
마우스 오른쪽 클릭: 조준
E: 인벤토리 열기
Q: 맵 열기
F: 가까이 있는 아이템을 줍는다.
SPACE BAR: NPC와 물건 교환, 상자 속의 아이템을 줍는다.


페이즈가 바뀔때 바다의 수면이 올라오는 것은 구현하지 못함.

서버 - VoteFight_IOCP.exe
클라이언트 - VoteFight.exe

게임이 시작된 후 서버 재접속 불가능
현재 클라이언트가 (10, 0, 10) (20, 0, 10) (30, 0, 10) 으로 배치되어 있는데, 이는 테스트를 위함.
실제 게임에서는 맵에서 가장 멀리 (10, 0, 10), (400, 0, 10) (400, 0, 400) 에 배치 될 예정
