이동 : WASD
마우스 왼쪽 클릭: 공격
마우스 오른쪽 클릭: 조준
E: 인벤토리 열기
Q: 맵 열기
F: 가까이 있는 아이템을 줍는다.
SPACE BAR: NPC와 물건 교환, 상자 속의 아이템을 줍는다.

서버 - VoteFight_IOCP.exe
클라이언트 - VoteFight.exe

====== 추가 =======
서버 재접속 가능 (플레이어가 모두 나가면 서버가 자동으로 초기화됨)
서버 인원 조정 가능 (1명일때 왼쪽 득표수 안나옴)
득표수 서버 연동
바다 높이보다 아래에 있으면 체력 깎임
Login Scene, Select Scene 배경 추가
총알 수, 장전 추가 (총을 들어야 총알 수가 나옴)
이동 애니메이션 버그 수정
ESC 누르고 Exit 버튼을 눌러야 게임이 꺼짐
서버 디버깅 지움 (최소한만 남김)
사운드 추가 (소리주의) (아직 별로 안 넣음. 더 추가 예정)
가파른 지형은 올라가지 못하도록 수정
총알은 갖고 있는 총알 수 만큼만 가질 수 있음.
아이템 합치기 가능 (오른쪽 마우스를 누른채 드래그 하면 1개씩 옮길 수 있음)
